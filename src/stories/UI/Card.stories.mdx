# Card

Cards are surfaces that display content and actions on a single topic.

They should be easy to scan for relevant and actionable information. Elements, like text and images, should be placed on them in a way that clearly indicates hierarchy.

<Source
  language="tsx"
  code={dedent`
    import { Card, CardHeader, CardMedia, CardContent, CardActions } from "@actionishope/shelley-cms";
  `}
/>

## Example

The following example uses a `CardHeader` and `CardContent` in conjunction with the `onPress` event to create a simple clickable card for adding a block.

<Story
  name="a) Card"

    docs: {

<Card onPress={() => console.log("onPress fired")}>

    <H2 vol={1} uppercase>
      Add Block
    </H2>
    <Add />
  </CardHeader>
  <CardContent>
    <P vol={1}>A simple, yet effective text only description here.</P>
  </CardContent>
</Card>`,
      },

  }}
>

</Story>
